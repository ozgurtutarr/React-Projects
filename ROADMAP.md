# 100 React Projects Roadmap

This roadmap is designed to take you from a React beginner to an expert.

## Level 1: Beginner (Foundations)

_Focus: JSX, Components, Props, State, Events, Conditional Rendering, Lists_

1.  **Hello World & Simple JSX**: Display a dynamic greeting.
2.  **Simple Counter**: Increment/decrement a number.
3.  **User Profile Card**: Static component with props (name, image, bio).
4.  **Color Toggler**: Change background color on button click.
5.  **List Renderer**: Render a list of items from an array.
6.  **Accordion**: Show/hide content sections.
7.  **Image Slider (Manual)**: Next/Prev buttons to switch images.
8.  **Digital Clock**: Update time every second (introduction to `useEffect` basics).
9.  **Stopwatch**: Start, stop, reset functionality.
10. **Random Quote Generator**: Display a new quote from an array on click.
11. **BMI Calculator**: Input height/weight, calculate result.
12. **Temperature Control**: Increase/decrease temp, change color based on value.
13. **Character Counter**: Input text, show character count.
14. **ToDo List (Basic)**: Add and delete items (no persistence).
15. **Hex Color Generator**: Generate random hex code and background.
16. **Search Filter**: Filter a list of names based on input.
17. **Simple Registration Form**: Uncontrolled inputs, alert data on submit.
18. **Accordion (Multi-select)**: Allow multiple sections open at once.
19. **Star Rating**: Hover and click to set rating.
20. **Dice Roller**: Random number 1-6 with dice icon.
